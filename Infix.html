e2

Description

There is no formal class definition for S3. Simply add 'Infix' or 'Print' to the class attribute and it inherits the methods. It is the same as Binary-class or Show-class just for S3. This is inteded to be used with retList.

Details

The lhs is coerced with as.environment and in that environment the binary operators must be found and named as . (see the example for retList). This is implemented for the following operators: +, -, *, /, %%, ^, <, >, ==, >=, <=, &. Also part of the operators you can implement with Infix is !, although it is unary.
